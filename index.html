Minha Biblioteca: Uma Webpage Personalizada

Os alunos criaram suas redes sociais no papel que se chama prototipo de baixa qualidade para vizualizarem como ficaria o prototipo para posteriormente ser passado para o figma. O objetivo do projeto era criar uma rede social, onde cada um pudesse desenvolver de maneira criativa e diversa.

O projeto da rede social foi realizado para que os alunos praticassem o uso da plataforma do figma. A rede social My Pets é como um instagram para pets, onde você pode seguir seus amigos e interagir com as fotos e vídeos de seus pets, utilizando cores para chamar a atenção das pessoas que utilizarem o app.

As cores utiliadas foram o rosa e o verde, rosa simbolizando as pets fêmeas e o verde os machos, essas cores também foram escolhidas para dar contraste para as pessoas com dificuldades visuais facilitando para o uso do app, as letras também utilizando fonte grande pensadas nisso também.

Robótica: projeto de leds, semáfaro de trânsito de carros e pedrestes

Os projetos de robótica é feito para que os alunos aprendam a mexer em tecnologia que utiliza fios e leds, entenam a função de resistores e cada peça, que consigam achar o problema quando há, e o mais importante que é o trabalho em equipe.

O projeto escolhido é o projeto de robótica paraná semáfaro de trânsito de carros e pedrestes, com o intuito dos alunos entenderem como funciona o semáfaro, leds e resistores, os alunos aprendem a trabalhar em equipe e quando o projeto der erro encontralo.

Com esse projeto podemos entender como o semáfaro funciona, quando a luz vermelha dos carros acende a luz verde dos pedrestes acende também, assim simbolizando que os carros param e as pessoas podem avançar.

Projeto de Robótica: Código morse

O projeto é sobre código morse, onde os materiais utilizados são: placa arduino, placa protoboard, LED, jumpers, resistor, push-button, buzzer, cabo USB, software, notebook. Esses são os materiais utilizados juntamente com os códigos no arduino editor.

Com esse projeto os alunos aprendem a montar os projetos e podem aprender a utilizarem os códigos morse, escrevendo uma frase e depois ouvir o som do buzzer pronunciando a frase em "bips", respondewndo com os códigos letra por letra.

A importância do projeto é que os alunos se divertem fazendo atividades em grupos, evoluindo o conhecimento e aprendento códigos, e entendendo como as coisas funcionam como nos avioẽs que utilizam códigos morse para transmitir mensagens.
